
🚀 AI Resume Analyzer & Job Recommendation System

📌 Overview
This project is an AI-powered Resume Analyzer integrated with a Job Recommendation System. It helps candidates analyze their resumes and match them with relevant job opportunities by using web scraping and artificial intelligence techniques.
The system collects job data from multiple online portals and compares it with candidate resumes to provide personalized recommendations and insights.

🎯 Features
📄 AI-based Resume Analysis
🔍 Skill Extraction from Resume
🌐 Web Scraping of Job Portals
🤖 Job Recommendation System
📊 Data Processing and Matching
💻 User-friendly interface using Streamlit
🛠️ Technologies Used
Python
BeautifulSoup (Web Scraping)
Selenium (Dynamic Scraping)
Pandas & NumPy (Data Processing)
API (AI Resume Analysis)
Streamlit (Web App Deployment)
MongoDB (Database)
⚙️ How It Works
Web Scraping Module
Extracts job data such as title, company, skills, and description from job portals.
Resume Analyzer
Upload resume
Extracts key information (skills, experience, education)
Uses AI to analyze and score the resume
